## 词根一：-auth-（作者 → 权威 → 授权）

### 📖 词根故事

```
auth- = 创始、作者
作者创造了作品 → 他有权威 → 他可以把权力授权给他人
```

> 一条线：author(作者) → authority(权威) → authorize(授权) → authentic(原创的=真实的)

### 🔗 单词家族

| 单词 | 释义 | 记忆线索 |
|------|------|------|
| **author** /ˈɔːθər/ | n. 作者 | 源头词，一切从这里来 |
| **authority** /əˈθɔːrəti/ | n. 权威；权力；当局(复数) | author+ity→作者有的东西 |
| **authoritative** /əˈθɔːrəteɪtɪv/ | adj. 权威性的；命令式的 | authority+ive |
| **authorize** /ˈɔːθəraɪz/ | v. 授权；批准 | authorise 同义（-ize/-ise） |
| **authorization** /ˌɔːθərəˈzeɪʃ(ə)n/ | n. 授权（书）；批准 | authorisation 同义 |
| **unauthorized** /ʌnˈɔːθəraɪzd/ | adj. 未经授权的 | unauthorised 同义 |
| **authentic** /ɔːˈθentɪk/ | adj. 真正的；真实的；原作的 | auth(创始)+entic→原创的→真的 |
| **authenticate** /ɔːˈθentɪkeɪt/ | v. 鉴定；证明是真的 | authentic+ate→使之真实 |

### 🧩 记忆口诀

```
author 是作者，authority 有权势
authorize 去授权，authentic 才是真
un 开头表否定，unauthorized 没被批
```

### 📝 必背例句

> **authorize**: We **authorize** the president to use force if necessary. 我们授权总统在必要时使用武力。

> **authentic**: He got an **authentic** letter by Shakespeare. 他弄到了一封莎士比亚的亲笔信。

## 词根一：-able-（能够）

### 📖 词根故事

```
able = 有能力的、能够的
```

这个词根本身就是一个完整的单词 `able`（能够的）。所有含 `-able-` 的词都和"能不能、能力"有关。

### 🔗 单词家族（一张表全记住）

| 单词 | 拆解 | 释义 | 记忆线索 |
|------|------|------|------|
| **able** /ˈeɪb(ə)l/ | 本身就是 | adj. 能够的 | 基础词，如 I am able to do it. |
| **enable** /ɪˈneɪb(ə)l/ | en(使) + able | v. 使能够 | en = 使 → 让你变得"能够" |
| **disable** /dɪsˈeɪb(ə)l/ | dis(否定) + able | v. 使残疾；使丧失能力 | dis = 不 → 不让你"能够" |
| **disabled** /dɪsˈeɪb(ə)ld/ | disable + ed | adj. 残疾的 | 被剥夺了能力的 |
| **disability** /ˌdɪsəˈbɪləti/ | dis + ability | n. 残疾；无能 | ability = 能力，dis + ability = 没能力 |
| **unable** /ʌnˈeɪb(ə)l/ | un(不) + able | adj. 不能的 | un = 不，unable = not able |

### 🧩 记忆口诀

```
able 是"能"，enable 使他能
dis 一否定，disable 就残废
un 也表否定，unable 不能做
ability 是能力，disability 没能力
```

### 📝 必背例句

> **enable**: His wits **enable** him to get a higher position. 他的才智使他能够晋升。

> **disable**: The accident may **disable** him for work. 这起事故可能会使他丧失工作能力。

## 词根二：-ab-（否定 / 偏离）

### 📖 词根故事

```
ab- = away from / not  → 偏离、否定、不正常
```

`ab-` 来自拉丁语，表示"离开、偏离"，引申为"不正常的、坏的"。"滥用"就是"不正常地使用"；"荒唐的"就是"偏离正常的"。

### 🔗 单词家族

| 单词 | 拆解 | 释义 | 记忆线索 |
|------|------|------|------|
| **abolish** /əˈbɑːlɪʃ/ | ab + ol(生长) + ish | v. 废除、取消 | "不让继续长下去"→ 废除制度 |
| **absurd** /əbˈsɜːrd/ | ab + surd(聋的) | adj. 荒唐可笑的 | "偏离到听不见道理"→ 荒唐 |
| **abuse** /əˈbjuːs/ | ab(偏离) + use(使用) | v./n. 滥用；虐待；辱骂 | 偏离正常使用 = 滥用 |
| **mistreat** /ˌmɪsˈtriːt/ | mis(错误) + treat(对待) | v. 虐待 | 错误对待 = 虐待（= abuse） |

> **distinguish**: abolish（废除制度） vs cancel（取消约会/计划）
> - abolish the old system 废除旧制度
> - cancel a meeting 取消会议

### 📝 必背例句

> **absurd**: Don't be **absurd**! 别荒唐了！

> **abuse**: Too many people **abuse** alcohol. 有许多人酗酒。

> **mistreat**: It's against the law to **mistreat** an animal. 虐待动物是违法的。

## 词根三：-academy-（学院 / 学术）

### 📖 词根故事


### 🔗 单词家族

|------|------|------|
| **academy** /əˈkædəmi/ | n. 学院；研究院；学会 | 基础词 |
| **academic** /ˌækəˈdemɪk/ | adj. 学术的；学院的 n. 大学教师；学者 | academy + ic → 和学院相关的 |
| **academically** /ˌækəˈdemɪkli/ | adv. 学术上；理论上 | academic + ally → 副词 |
| **academician** /əˌkædəˈmɪʃ(ə)n/ | n. 院士 | academy + ian(人) → 学院里的人 |

## 词根四：-act-, -ag-（行动 / 做）★ 重点

### 📖 词根故事

```
act = 行动、做、扮演
ag  = act 的变体

这两个是一个父亲生的：拉丁语 agere（做、驱动）
act → 行动本身；ag → 行动的人/机构
```

本组是 Day 01 最大的词根组，单词最多，但**只要抓住 act = "行动/做"这个核心**，所有派生词都有迹可循。

### 🔗 act 核心家族矩阵

```
                    ┌─ action (n.)        行动、行为
                    ├─ active (adj.)       积极的、主动的
                    ├─ actively (adv.)     积极地
                    ├─ activist (n.)       积极分子
act (行动/做)  ────├─ activate (v.)       激活、使活动
                    ├─ activity (n.)       活动
                    ├─ actual (adj.)       真实的、实际的 ← 做出来的才算真的
                    ├─ actually (adv.)     实际上
                    ├─ react (v.)          反应 ← re(回) + act = 回应行动
                    ├─ interact (v.)       互动 ← inter(之间) + act = 互相作用
                    ├─ exact (adj.)        精确的 ← ex(出) + act = 做出来分毫不差
                    ├─ enact (v.)          制定(法律)；扮演 ← en(使) + act(法案)
                    ├─ agent (n.)          代理人 ← ag(做) + ent(人) = 做事的人
                    ├─ agency (n.)         机构、代理处
                    └─ agenda (n.)         议程 ← 要做的事
```

### 🧩 分组记忆

**① 基础链：act → action → active → activity**

```
act(行动) → action(行动/行为，名词化)
          → active(行动多的 → 积极的)
          → activity(活动，行为的总称)
```

**② 真实链：act → actual → actually**

```
"做出来的才算真的"——通过行动证明的才是真实的
act + ual(形容词后缀) = actual 真实的
actual + ly = actually 实际上
```

**③ 互动链：react / interact / exact**

```
re(回) + act = react     回应动作 → 反应
inter(之间) + act = interact  互相作用 → 互动
ex(出，彻底) + act = exact  做得彻底精准 → 精确的
```

**④ 代理人链：ag(做) → agent → agency → agenda**

```
ag(做) + ent(人) = agent     做事的人 → 代理人
agent + cy(处所) = agency     代理人办公处 → 机构
ag(做) + enda(应做) = agenda  应该做的事 → 议程
```

### 📝 必背例句

> **active**: She is an **active** participant in this activity. 她积极参与这项活动。

> **actual**: The **actual** price was lower than I thought. 实际价钱比我想象的便宜。

> **react**: Try not to **react** in anger. 不要在愤怒中作出回应。

> **interact**: Partners must **interact** with each other's ideas. 合作伙伴必须相互交流想法。

> **enact**: States are free to **enact** stricter regulations. 各州可自由制定更严格的规则。

> **agent**: He works as a house **agent**. 他是一名房产经纪人。

### ⑤ 痛苦链：agitate / agony

`ag-` 的另一条线：反复搅动 → 心神不宁 → 痛苦

| 单词 | 释义 | 记忆线索 |
|------|------|------|
| **agitate** /ˈædʒɪteɪt/ | v. 煽动；搅动；使焦虑 | ag(做/驱动) + itate → 反复驱动 → 煽动 |
| **agitator** /ˈædʒɪteɪtər/ | n. 煽动者；搅拌器 | agitate + or(人/物) |
| **agony** /ˈæɡəni/ | n. 极大的痛苦 | agon(挣扎) + y → 痛苦的挣扎 |
| **agonize** /ˈæɡənaɪz/ | v. 苦苦思索；使极度痛苦 | agony + ize(动词化) |

# 同位语从句

## 📖 怎么理解同位语从句

同位语从句就是用一个完整的句子来解释一个**抽象名词**的具体内容。这个名词和从句是"同一位"的关系——从句 = 名词的内容。

> 画面感：同位语从句像一个"标签展开"——先给一个概念词，然后用一个句子把这个概念的内容展开说明。
>
> - The **news** that he won surprised us. → "他赢了"这条消息
> - The **fact** that he lied is obvious. → "他说谎了"这个事实
>
> `that he won` 和 `the news` 指的是**同一件事**。

---

## 一、同位语从句的基本结构

```
抽象名词 + that + 完整陈述句
```

| 常见抽象名词 | 含义 |
|------|------|
| fact | 事实 |
| news | 消息 |
| idea | 想法、观点 |
| hope | 希望 |
| belief | 信念 |
| thought | 想法 |
| truth | 真相 |
| suggestion | 建议 |
| promise | 承诺 |
| conclusion | 结论 |
| opinion | 观点 |
| evidence | 证据 |
| possibility | 可能性 |

### 例句

| 例句 | 从句 = 名词的内容 |
|------|------|
| The **fact that he lied** is obvious. | "他说谎了" = 这个事实 |
| I heard the **news that he had won**. | "他赢了" = 这条消息 |
| The **hope that he would recover** kept us going. | "他会康复" = 这个希望 |
| He made a **promise that he would come back**. | "他会回来" = 这个承诺 |

---

## 二、同位语从句 vs 定语从句 ★ 考研必考

这是考研完形填空和阅读的高频辨析点！

| | 同位语从句 | 定语从句 |
|------|------|------|
| **that 的作用** | 只起**连接作用**，不充当成分 | 在从句中**充当主语或宾语** |
| **从句是否完整** | **完整**（不缺成分） | **不完整**（缺主语或宾语） |
| **修饰什么** | 抽象名词的**内容** | 名词的**特征/范围** |

### 核心判断法：看 that 在从句中是否充当成分

```
The news that he told me is true.          → 定语从句
              ↓
         he told me the news → that 是 told 的宾语 → 充当成分

The news that he won is true.              → 同位语从句
              ↓
         he won → 从句完整，that 无职位 → 不充当成分
```

### 更多对比

| 同位语从句 | 定语从句 |
|------|------|
| The **fact that** the earth is round is true. | The **fact that** he mentioned is true. |
| 地球是圆的这个事实是真的。 | 他提到的那个事实是真的。 |
| that 在从句中无职位（从句完整） | that 是 mentioned 的宾语（缺宾语） |
| The **suggestion that** we should leave now is good. | The **suggestion that** he put forward is good. |
| 我们现在就走的建议很好。 | 他提出的建议很好。 |

> 判定口诀：**that 后面跟的是完整句 → 同位语；跟的是缺东西的句 → 定语**

## 三、whether / what / how 等引导的同位语从句

同位语从句不只用 that，也可以用其他引导词：

| 引导词 | 例句 |
|------|------|
| **whether** | The question **whether we should go** remains. |
| **what** | I have no idea **what he wants**. |
| **when** | Do you have any idea **when he will arrive**? |
| **where** | The problem **where we should meet** is unresolved. |
| **why** | He couldn't explain the reason **why he was late**. |
| **how** | The question **how we can solve it** is important. |

### have no idea + 同位语从句

```
I have no idea what you're talking about.
         └────── 同位语从句 ──────┘
= 我不知道你在说什么。
```

```
I have no idea when he will come.
= 我不知道他什么时候来。
```

> have no idea = don't know，后面跟的是同位语从句，说明"idea"的内容。

## 四、分隔式同位语从句

同位语从句不一定紧跟在名词后面，可能被谓语隔开：

```
The news spread quickly that he had won the prize.
└─主语─┘  └谓语─┘  └状语─┘  └──── 同位语从句 ────┘
                                    |
                            修饰 the news（被 spread quickly 隔开了）
```

> 谓语太短，同位语从句太长 → 为了平衡，从句被推到句末。这是考研长难句的常见结构。

| 正常 | 分隔式 |
|------|------|
| The **news that he had won** spread. | The **news** spread quickly **that he had won**. |

---

## 五、同位语从句中的虚拟语气

部分抽象名词后面的同位语从句要用 **虚拟语气 (should) do**：

| 常见名词 | 例句 |
|------|------|
| suggestion 建议 | The **suggestion** that he **(should) leave** was accepted. |
| proposal 提议 | The **proposal** that we **(should) start** early is good. |
| requirement 要求 | The **requirement** that everyone **(should) attend** is strict. |
| order 命令 | The **order** that the soldiers **(should) retreat** came. |
| advice 建议 | His **advice** that I **(should) wait** proved right. |

> 这类名词来自对应的动词（suggest → suggestion, require → requirement），保留了动词的虚拟语气用法。

---

## 六、同位语 vs 主语补足语

同位语不一定是从句，也可以是一个词或短语：

| 形式 | 例句 |
|------|------|
| 名词作同位语 | Mr. Wang, **our teacher**, is very kind. |
| 短语作同位语 | He has only one hobby, **reading novels**. |
| 从句作同位语 | The fact **that he lied** is obvious. |

## 七、练习

### 一、判断划线部分是什么从句（定语从句 or 同位语从句）

1. The fact that he told me is true. 定语
2. The fact that he is honest is true. 同位语
3. The news that he brought was exciting. 定语
4. The news that he had won was exciting. 同位语
5. I have no idea what he is talking about. 同位语

### 二、翻译

1. 他迟到的消息让我们很惊讶。
2. 我完全不知道他是谁。
3. 他做出了一个以后会更努力的承诺。
4. 关于他是否会来，这个问题仍然没有答案。
5. 他通过考试这个事实让所有人高兴。

### 三、改错

1. The fact which he lied is obvious.
2. I have no idea that he is talking about.
3. The suggestion that we start earlier was accepted.（这句没错，思考为什么）

## 参考答案

### 一、判断
1. **定语从句**（he told me the fact → that 作宾语）
2. **同位语从句**（he is honest → 从句完整，fact = he is honest）
3. **定语从句**（he brought the news → that 作宾语）
4. **同位语从句**（he had won → 从句完整，news = he had won）
5. **同位语从句**（what he is talking about = idea 的内容）

### 二、翻译
1. The **news that he was late** surprised us.
2. I have **no idea who he is**.
3. He made a **promise that he would work harder**.
4. The **question whether he will come** still remains unanswered.
5. The **fact that he passed the exam** made everyone happy.

### 三、改错
1. ❌ which → ✅ **that**（同位语从句用 that 引导，不能用 which）
2. ❌ that → ✅ **what**（what 在从句中作 talking about 的宾语）
3. ✅ 正确。suggestion 后面的同位语从句用 should do 虚拟语气（should 可省略），start 是动词原形。

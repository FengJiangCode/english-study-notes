# 表语从句

## 📖 怎么理解表语从句

表语从句就是**系动词后面用一个完整的句子来当表语**，说明主语"是什么、怎么样"。

> 回忆：系动词 = be 动词 + 感官类（look/sound/smell）+ 变化类（become/get）+ 保持类（remain/stay）
>
> - 简单表语：The problem is **money**. 问题是钱。
> - 表语从句：The problem is **that we have no money**. 问题是我们没有钱。

---

## 一、表语从句基本结构

```
主语 + 系动词 + 引导词 + 陈述句从句
```

| 例句 | 系动词 |
|------|------|
| The truth is **that he lied**. | is |
| That is **what I want**. | is |
| The question remains **whether we should go**. | remains |
| He looks **as if he had seen a ghost**. | looks |

---

## 二、that 引导的表语从句

### 基本用法

| 主语 | 例句 |
|------|------|
| the truth / fact is | The truth is **that he never loved her**. |
| the problem is | The problem is **that we don't have enough time**. |
| the reason is | The reason is **that he was ill**. |
| the good news is | The good news is **that everyone passed**. |
| my opinion / view is | My view is **that we should wait**. |

### that 在表语从句中不能省略

> ✅ The truth is **that** he lied.
> ❌ The truth is he lied.（口语也许能接受，考研写作和语法考试中算错）

---

## 三、because 表语从句 ★ 

**It/This/That is because... 是英语中非常常用的表达：**

| 例句 | 翻译 |
|------|------|
| He was late. It was **because he missed the bus**. | 他迟到了。那是因为他错过了公交车。 |
| That is **because you didn't work hard**. | 那是因为你没努力。 |

### This is why... vs This is because...

| | This is why... | This is because... |
|------|------|------|
| 含义 | 这就是……的原因（结果→原因） | 这是因为……（原因解释） |
| | He missed the bus. **That is why** he was late. | He was late. **That is because** he missed the bus. |
| | （没赶上公交→所以迟到） | （迟到→因为没赶上公交） |

---

## 四、whether 引导的表语从句

whether 在表语从句中表示 **"是否"**，保留"不确定性"。

| 例句 | 翻译 |
|------|------|
| The question is **whether he will come**. | 问题是他是否会来。 |
| The problem remains **whether we can afford it**. | 问题仍然是我们能否负担得起。 |

> 表语从句中**不能用 if**：
> ✅ The question is **whether** he will come.
> ❌ The question is **if** he will come.

---

## 五、连接代词引导的表语从句

连接代词 what / who / which 等引导，在从句中充当成分：

| 例句 | 引导词在从句中的职位 |
|------|------|
| That is **what I want**. | want 的宾语 |
| He is **who I was talking about**. | about 的宾语 |
| The question is **which is better**. | is 的主语 |
| This is **what worries me most**. | worries 的主语 |

### what 表语从句

```
This is what I mean.
       └─ what = the thing that，在从句中作 mean 的宾语

= This is the thing that I mean.
```

---

## 六、连接副词引导的表语从句

| 引导词 | 例句 | 从句中充当 |
|------|------|------|
| **when** | Tomorrow is **when we leave**. | 时间状语 |
| **where** | That's **where I grew up**. | 地点状语 |
| **why** | That's **why I came here**. | 原因状语 |
| **how** | This is **how it works**. | 方式状语 |

### This/That is why...（高频表达）

```
He studied hard. That is why he passed the exam.
他努力学习。那就是他通过考试的原因。

= He passed the exam because he studied hard.
```

### This/That is where/how...

```
That's where we disagree.   这就是我们意见不同的地方。
This is how I solved it.    我就是这么解决的。
```

---

## 七、as if / as though 引导的表语从句

感官系动词（look, sound, feel, seem）后面可以用 as if / as though：

| 例句 | 翻译 |
|------|------|
| It **looks as if** it is going to rain. | 看起来要下雨了。 |
| He **sounds as though** he knows everything. | 他听起来好像什么都知道。 |
| I **feel as if** I were dreaming. | 我感觉像在做梦。 |

---

## 八、The reason is that... 考点

```
The reason (why...) is that...    ← 表语从句用 that，不用 because

✅ The reason why he was late is that he missed the bus.
❌ The reason why he was late is because he missed the bus.
```

> reason 作主语时，表语从句用 **that** 引导，不用 because。这是考研改错的高频陷阱。

---

## 九、主语从句 vs 表语从句

有时同一个意思可以用两种方式表达：

| 主语从句 | 表语从句 |
|------|------|
| **What he said** is true. | The truth is **what he said**. |
| **Whether he comes** is the question. | The question is **whether he comes**. |
| **Why he left** is a mystery. | The mystery is **why he left**. |

> 主语从句 → 把"那件事"放句首；表语从句 → 先出主语名词，从句放后面补充。意思一样，表达习惯不同。

## 十、练习

### 一、选择合适的引导词

1. The problem is ________ we don't have enough time. (that / what / if)
2. That's ________ I disagree with you. (where / which / what)
3. The question is ________ we should go or stay. (if / whether / that)
4. He studied hard. That's ________ he passed the exam. (because / why / how)
5. This book is just ________ I've been looking for. (what / that / which)

### 二、翻译

1. 问题是我们没有足够的钱。
2. 这就是我不同意的地方。
3. 他迟到了。那是因为他起晚了。
4. 问题仍然是他是否会来。
5. 他看起来好像生病了。

### 三、改错

1. The reason is because he missed the bus.
2. The question is if he will come.
3. That's that I want to say.

---

## 参考答案

### 一、填空
1. **that**（从句完整，用 that 连接）
2. **where**（where = 在……的地方，从句中作 disagree 的地点状语）
3. **whether**（or stay → 用 whether；表语从句不能用 if）
4. **why**（That's why = 那就是为什么/原因）
5. **what**（what = the thing that，从句中作 looking for 的宾语）

### 二、翻译
1. The problem is **that** we don't have enough money.
2. That's **where** I disagree.
3. He was late. That's **because** he got up late.
4. The question remains **whether** he will come.
5. He looks **as if/as though** he is sick.

### 三、改错
1. ❌ because → ✅ **that**（reason 作主语，表语从句用 that）
2. ❌ if → ✅ **whether**（表语从句不能用 if）
3. ❌ that → ✅ **what**（what = the thing that，作 say 的宾语）

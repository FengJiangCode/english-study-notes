# 倒装句

## 📖 怎么理解倒装

倒装就是把句子的**正常语序颠倒过来**。英语正常语序是"主语 + 谓语"，倒装就是"谓语（的一部分）跑到主语前面"。

> 画面感：正常语序像排队——主语在前面，谓语在后面。倒装就像有人插队——谓语的一部分跑到了主语前面。
>
> - 正常：He **is** a teacher. 主语→系动词→表语
> - 倒装：**Is** he a teacher? 系动词→主语→表语（一般疑问句就是最简单的倒装）

---

## 一、全部倒装 vs 部分倒装

| | 全部倒装 | 部分倒装 |
|------|------|------|
| **规则** | 整个谓语提到主语前 | 只把助动词/情态动词提到主语前 |
| | Here **comes the bus**. | **Never have I seen** such a thing. |
| | 公交车来了。 | 我从没见过这样的事。 |

---

## 二、部分倒装 ★ 考研重点

### 公式：把助动词/情态动词提到主语前面

```
正常：I have never seen such a thing.
倒装：Never have I seen such a thing.
      ↑ 否定词提前 → 助动词 have 提到主语 I 前
```

### 六大触发条件

**1. 否定词/半否定词位于句首**

| 触发词 | 例句 |
|------|------|
| never | **Never** have I felt so happy. |
| seldom | **Seldom** does he go out. |
| hardly | **Hardly** had I arrived when it rained. |
| scarcely | **Scarcely** had he left when she called. |
| not only...but also | **Not only** is he smart, **but** he is also kind. |
| no sooner...than | **No sooner** had we sat down than the show began. |
| under no circumstances | **Under no circumstances** should you lie. |

**2. only + 状语位于句首**

```
Only then did I understand.         Only + 时间副词
Only in this way can we succeed.    Only + 介词短语
Only when he left did I cry.        Only + 状语从句
```

> ⚠️ only 修饰主语时**不倒装**：Only **Tom** knows the answer. ✅（Tom 是主语）

**3. so / such...that 结构中 so/such 位于句首**

```
So fast did he run that nobody could catch him.
Such a beautiful day was it that we went out.
```

**4. 虚拟语气中 if 省略（were/had/should 提前）**

```
If I were you → Were I you, I would go.
If he had known → Had he known, he would have come.
If it should rain → Should it rain, we'll cancel.
```

**5. as / though 引导的让步状语从句**

```
Young as he is, he knows a lot.    虽然他很年轻，但他懂得很多。
Try as he might, he failed.        尽管他试了，还是失败了。
```

**6. neither / nor / so 表示"也"**

```
He can't swim. Neither can I.      他不会游泳。我也不会。
She likes coffee. So do I.         她喜欢咖啡。我也喜欢。
```

---

## 三、全部倒装

### 触发条件

**1. 地点状语位于句首（here, there, in, out, up, down, away...）**

```
Here comes the bus.        公交车来了。
There goes the bell.       铃响了。
Away flew the birds.       鸟儿飞走了。
```

> ⚠️ 主语是**代词**时不倒装：Here **it** is. ✅（不是 Here is it.）

**2. 表语/分词位于句首**

```
Standing at the door was a policeman.     站在门口的是一名警察。
Gone are the days when we were poor.      我们贫穷的日子一去不复返了。
```

---

## 四、倒装句 vs 强调句 vs 形式主语

| 句式 | 特征 | 例句 |
|------|------|------|
| **倒装** | 谓语跑主语前 | Never have I seen it. |
| **强调** | It is...that 框架 | It was I that saw it. |
| **形式主语** | It 代指后面的 that 从句 | It is clear that he lied. |

## 五、练习

### 一、将下列句子改写为倒装句

1. I have never seen such a beautiful sunset.
2. He not only sings well, but he also dances well.
3. We can succeed only by working hard.
4. If I were you, I would accept the offer.
5. The bell goes there.

### 二、改错

1. Never I have been to Beijing.
2. Only then I realized my mistake.
3. Here the bus comes.
4. Not until he arrived did I left.

### 三、翻译

1. 我从没见过这么美丽的景色。
2. 只有用这种方法我们才能解决问题。
3. 虽然他很年轻，但他懂得很多。（用 as）
4. 如果我早知道，我就不会去了。（用 Had）

---

## 参考答案

### 一、改写
1. **Never have I seen** such a beautiful sunset.
2. **Not only does he sing** well, but he also dances well.
3. **Only by working hard can we** succeed.
4. **Were I you**, I would accept the offer.
5. **There goes the bell**.

### 二、改错
1. ❌ Never I have been → ✅ **Never have I been**（否定词句首，助动词提前）
2. ❌ Only then I realized → ✅ **Only then did I realize**（only + 状语，倒装）
3. ❌ Here the bus comes → ✅ **Here comes the bus**（地点状语句首，全部倒装；但主语是代词时不倒装，Here it comes ✅）
4. ❌ did I left → ✅ **did I leave**（did 后面用动词原形）

### 三、翻译
1. **Never have I seen** such a beautiful view.
2. **Only in this way can we** solve the problem.
3. **Young as he is**, he knows a lot.
4. **Had I known** earlier, I wouldn't have gone.

# 插入语

## 📖 怎么理解插入语

插入语是**独立于句子结构之外的成分**，用逗号、破折号或括号隔开，表示说话人的态度、看法、补充说明。去掉它，句子结构依然完整。

> 画面感：插入语就像你说话时突然"插一句"——不改变原句的语法，只是加了一层个人色彩。
>
> - He is, **I think**, the best candidate. 他是，我觉得，最好的候选人。
> - 去掉 I think：He is the best candidate. ✅ 句子完整

---

## 一、插入语的形式

| 形式 | 例句 |
|------|------|
| **单词** | It is, **indeed**, a good idea. 这确实是个好主意。 |
| **短语** | **To be honest**, I don't like it. 说实话，我不喜欢。 |
| **短句** | He is, **I believe**, trustworthy. 他是，我相信，值得信赖的。 |
| **破折号/括号** | My brother — **the one in Beijing** — is a doctor. |

---

## 二、插入语的六大类型

### 1. 表示说话人态度（最常见）

| 插入语 | 翻译 |
|------|------|
| **I think / I believe** | 我认为 |
| **I'm afraid** | 恐怕 |
| **I'm sure** | 我确定 |
| **I suppose** | 我想 |
| **I hope** | 我希望 |

> He will come, **I'm sure**. 他会来的，我确定。

### 2. 表示逻辑关系

| 插入语 | 翻译 |
|------|------|
| **however** | 然而 |
| **therefore** | 因此 |
| **meanwhile** | 与此同时 |
| **in addition** | 此外 |
| **for example / for instance** | 例如 |

### 3. 表示强调/评价

| 插入语 | 翻译 |
|------|------|
| **indeed** | 确实 |
| **in fact / as a matter of fact** | 事实上 |
| **undoubtedly** | 毫无疑问 |
| **obviously** | 显然 |
| **fortunately / unfortunately** | 幸运的是 / 不幸的是 |

### 4. 表示解释/补充

| 插入语 | 翻译 |
|------|------|
| **that is (to say)** | 也就是说 |
| **in other words** | 换句话说 |
| **or rather** | 更确切地说 |
| **namely** | 即 |

### 5. 表示信息来源

| 插入语 | 翻译 |
|------|------|
| **it is said** | 据说 |
| **it is reported** | 据报道 |
| **as we all know** | 众所周知 |
| **in one's opinion** | 在某人看来 |

### 6. 不定式/分词作插入语

| 插入语 | 翻译 |
|------|------|
| **to be honest** | 说实话 |
| **to be frank** | 坦白说 |
| **to tell you the truth** | 跟你说实话 |
| **to make matters worse** | 更糟糕的是 |
| **generally speaking** | 一般来说 |
| **frankly speaking** | 坦率地说 |
| **strictly speaking** | 严格来说 |
| **judging from...** | 从……来看 |

---

## 三、插入语 vs 常规状语

| | 插入语 | 状语 |
|------|------|------|
| **与句子关系** | 结构独立，用逗号隔开 | 和动词/句子有语法关联 |
| | **Fortunately**, he passed. | He spoke **frankly**. |
| | fortunately 是插入语（评价整个事件）| frankly 是状语（修饰 spoke 的方式）|

---

## 四、插入语在长难句中的处理

考研阅读中插入语会**打乱主谓结构**，读的时候先跳过：

```
The man, I thought, was my friend.
      └─ 插入 ──┘

读法：The man ... was my friend. 然后再把 I thought 插回去理解。
```

```
This policy, however, has been criticized.
             └── 插入 ──┘

读法：This policy ... has been criticized. 然而这项政策受到了批评。
```

## 五、练习

### 一、找出插入语并翻译

1. He is, in my opinion, the best choice.
2. To be honest, I don't agree with you.
3. The plan, however, was rejected.
4. This book, I think, is worth reading.
5. Unfortunately, the tickets were sold out.

### 二、翻译

1. 说实话，我不喜欢这个主意。
2. 据我所知，他明天会来。
3. 这个人，我相信，是可靠的。
4. 幸运的是，没有人受伤。

### 三、用插入语改写

1. I believe he is honest. → He is, **I believe**, honest.
2. It's true that we lost the game. → We lost the game, **indeed**.

## 参考答案

### 一、找插入语
1. **in my opinion** — 在我看来，他是最好的选择。
2. **To be honest** — 说实话，我不同意你。
3. **however** — 然而，这个计划被否决了。
4. **I think** — 这本书，我认为，值得一读。
5. **Unfortunately** — 不幸的是，票卖完了。

### 二、翻译
1. **To be honest / To tell you the truth**, I don't like this idea.
2. **As far as I know**, he will come tomorrow.
3. The man, **I believe**, is reliable.
4. **Fortunately**, nobody was hurt.

### 三、改写
1. He is, **I believe**, honest.
2. We lost the game, **indeed**.

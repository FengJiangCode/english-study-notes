# 考研英语学习项目

## 项目目标
系统化学习考研英语，覆盖七大模块：词汇、语法、长难句、阅读、完形填空、翻译、写作。

## 目录结构

| 目录 | 用途 |
|------|------|
| `词汇/` | 考研词汇记忆、词根词缀、核心词汇、高频词组 |
| `语法/` | 语法知识梳理（时态、语态、从句、非谓语动词、虚拟语气等） |
| `长难句/` | 长难句拆分与解析练习 |
| `阅读/` | 阅读理解训练，按题材分类（经济、科技、人文、社会等） |
| `完形填空/` | 完形填空专项训练 |
| `翻译/` | 英译汉翻译练习 |
| `写作/` | 大作文（图画作文）和小作文（应用文）模板与练习 |
| `真题/` | 历年真题试卷与解析 |
| `笔记/` | 各类学习笔记、错题整理、方法总结 |

## 学习阶段建议
1. **基础阶段**：词汇积累 + 语法梳理 + 长难句分析
2. **强化阶段**：阅读 + 完形填空 + 翻译 专项训练
3. **冲刺阶段**：写作模板 + 真题模考
